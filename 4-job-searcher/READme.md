# Job Search 

A responsive, vanilla JavaScript web application that allows users to search for real-time job listings in India. The application features a unique **"Technical Schematic" / Blueprint** design theme and persists user data locally.

## Features

* **Real-Time Search:** Fetches live job data using the **Adzuna API**.
* **Blueprint UI:** A custom, fully responsive CSS theme designed to look like an architectural blueprint or technical schematic.
* **Favorites System:** Save interesting jobs to a "Favorites" list (persisted via `localStorage` so data remains after refresh).
* **Responsive Grid:** Automatically adjusts layout from mobile (1 column) to ultra-wide screens (4+ columns).
* **Mock Data Fallback:** Includes a robust mock data generator that works immediately if API keys are missing or limits are hit.

## Tech Stack

* **HTML5** (Semantic structure)
* **CSS3** (Custom variables, CSS Grid, Flexbox, no frameworks)
* **JavaScript (ES6+)** (Async/Await, DOM Manipulation, LocalStorage API)
